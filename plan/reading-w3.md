## 1日目：意見文① — 文末表現と医療ミス

**ポイント：文末表現に注意！**
- **～一方だ**：～の状態だ。
- **～つつある**：正で最中。
- **～に至る**：～という結果になった。
- **～次第だ**：結果や理由を示す。
- **～始末だ**：悪い状態になった。

### 練習：会話文
男：また病院のミスで患者が死んだんだって。看護師が患者に間違った薬を点滴したらしいよ。
女：薬の確認は当然するでしょう？
男：確認はしたけど患者の名前が間違って書かれていたんだって。
女：信じられない。そんなケアレスミスで人の命が奪われるなんて…。
男：ひどいよね。ニュースにならないミスはもっともっとあるということなんだろうね。

**問（正しいものを選ぶ）**
1. 患者が死んだのは、病院のせいである。
2. 病院の不注意で、患者が違う薬を飲んでしまった。
3. 看護師は薬の確認をしなかった。
4. 女の人は看護師たちが薬の名前を知らなかったことが信じられない。
5. 男の人は、病院の不注意はたくさんあると思っている。

### 問題：増え続ける医療ミス
「増え続ける医療ミス」という投書の内容。病院のミスで患者が亡くなる事故が相次いでいる。今回のミスは似た名前の別人に誤った点滴をしたこと。院長が謝っても命は戻らない。病院でケアレスミスは許されない。

**問1** 「確認すれば」とあるが、何を確認するのか。
1. ケアレスミス 2. 患者の状態 3. 点滴の量 4. 患者の名前

**問2** 「人ごととはせずに」とあるが、どういう意味か。
1. 自分には関係ないことと思わないで
2. 人間だけのこととは思わないで
3. 患者だけのこととは思わないで
4. 重要ではないとは思わないで

## 2日目：意見文② — カタカナ語とマナー

**ポイント：和製英語（Japanese English）に注意！**
- モーニングコール、ケアレスミス、ペーパーテスト、ベビーカー、パソコン、リモコン、コンビニ、マナーなど。これらは外来語ではなく日本語として考えましょう。

### 練習：会話文
A：最近の若い親には頭に来るんですよ。電車の中で子どもが走っていても注意しないし、靴のまま座席に上がらせるし、しつけがなってませんよ。
B：しつけがなっていないどころか、親自身のマナーがひどい。注意するとにらんでくるんですよ。あのベビーカーもやめてほしい。混んだ電車にそのまま乗ってくるし。
A：親にも子どもにも注意してやらないといけないってことですね。

**問（正しいものを選ぶ）**
1. Aさんは自分の子どものしつけができていない。
2. Bさんは親より子どものしつけが大切だと言っている。
3. AさんもBさんも親のマナーが悪いと思っている。
4. Bさんはベビーカーに子どもを乗せたまま電車に乗ってくる親に文句がある。
5. Aさんはにらまれるので親には注意したくない。

### 問題：チョット一言（投書）
最近の若い母親に言いたい。混んだ電車の中ではベビーカーをたたんでほしい。赤ちゃんを乗せているのは大変だとわかるが、平気で混んだ車内に乗り込むのは迷惑。今のベビーカーは軽く簡単たためる。子どもをしつける前に、まず教育しなければならないのは親だ。

**問1** 「にらみつけてくる」ときの母親の気持ちに近いものは？
1. しつけがなっていない人だな。 2. やさしくない人だな。 3. 迷惑な人だな。 4. 気が強い人だな。

**問2** 空欄②に入る言葉は？
1. 混んだ電車ではベビーカーをたたんでください 2. 混んだ電車に乗らないでください 3. 子どもの前に親の教育をしてください 4. だまっていないで注意をしてください

## 3日目：意見文③ — 筆者の主張

**ポイント：筆者が最も言いたいことを見抜こう**
- 常識や一般的意見ではなく、筆者がいちばん言いたいことを読み取る。

### 練習：会話文
A：この辺も物騒になってきたね。コンビニ強盗があったし。
B：あの犯人の高校生、お金持ちで不自由していなかったらしいよ。
A：ただスリルを味わいたかったんだって。
B：そんな理由ってないんじゃないの？

**問（正しいものを選ぶ）**
1. 最近この近くは危険な出来事が多い。
2. コンビニ強盗の犯人はまだ捕まっていない。
3. 犯人の高校生はお金持ちの家に強盗に入った。
4. 犯人はお金が目的でコンビニ強盗をした。
5. 犯人はスリルを楽しむためにコンビニ強盗をした。

### 問題：盗人にも三分の理
昔は「盗人にも三分の理」という言葉があったが、近ごろの犯罪はどうだろう。不自由のない普通の少年少女が、退屈だからと物を壊したり、スリルを楽しむために強盗をする。被害者を人間扱いせず、効率のみを考えたやり方で、理などどこにもない。

**問1** 筆者が近ごろの犯罪について最も言いたいことは？
1. 昔と違い、理由のない犯罪が多い。 2. お金が目的ではない。 3. 強盗や殺人が多い。 4. 普通の若者が多い。

**問2** 空欄②に入る言葉は？
1. 1%くらいはあると言えるだろう 2. 1%くらいは見つかるかもしれない 3. まったく見つからないのである 4. まったくないとは言えないのである

## 4日目：説明文① — 指示語と焦点を当てる読み方

**ポイント：問いから読んで、早く答えを探す**

### 練習：会話文
妻：あの子、毎日だらだら過ごして…パパからも叱って。若い時は二度ないのに。
夫：あいつはあいつで考えているさ。君はどうなの？ 毎日忙しく過ごせばいいってわけじゃない。若い時だけが大事じゃない。人生にはその年で特有の喜びがあるんだ。俺は今をただ老後の準備にはしてほしくないね。

**問（正しいものを選ぶ）**
1. 妻は若い人は若い時を大事に過ごすべきだと思っている。
2. 夫は大事に過ごすのは中年も老年も同じだと思っている。
3. 妻は老後をどう過ごすかが大事だと思っている。
4. 夫は子どもにあまり勉強してほしくないと思っている。
5. 妻は子どもより自分と夫の老後のことを心配している。

### 問題：若い時は二度ない
「若い時は二度ない」と言うが、中年も老年も一度しかない。若い時だけ特別視する必要はない。それぞれの段階に特有の人生のこくがある。人々は若い時を準備段階と考えるが、若い時に特有の喜びや悲しみを味わわないと、中年・老年になっても人生が味わえない。

**問1** 「若い時代だけを特別視する必要はない」のはなぜ？
1. 老年のほうが人生のよさがあるから。 2. 若い時代と同様に、中年、老年も大事だから。 3. 自分が皮肉屋だから。 4. 若いときは一度しかないから。

**問2** 「それはまちがいだと思う」が、何がまちがい？
1. 若いうちから遊びほうけること 2. 若い時代にしっかりと学問や体験を積むこと 3. 若い時代を特別視しないこと 4. 若者に自制と禁欲を呼びかけること

## 5日目：説明文② — 指示語に注意

**ポイント：指示語が指す範囲に注意！**
- 文だけでなく、長い内容を指すこともある。

### 練習：会話文
男：ここも禁煙か。吸える場所が減って居心地悪いよな。
女：やめちゃえば？値上がりしてるし、体に悪いし、周りにも迷惑よ。
男：でも、だったら酔っ払いだって迷惑だよ。
女：いえ、お酒より絶対に迷惑よ。煙は体に入ってくるんだから。

**問（正しいものを選ぶ）**
1. この会社では、タバコを吸う場所がない。
2. 男の人は道路でタバコを吸って罰金を払った。
3. 女の人は男の人にタバコをやめるように勧めている。
4. 男の人は、喫煙者は酔っ払いほど迷惑をかけていないと言っている。
5. 女の人は、酔っ払いより喫煙者のほうが迷惑だと思っている。

### 問題：近年の禁煙
日本では「分煙」「終日禁煙」が当たり前になっている。喫煙者は「なぜ酒はよくてタバコはダメなのか」と反論するが、酒が周囲に与える影響と違い、タバコの煙は周囲の肺に入る。これが喫煙者が嫌われる大きな理由である。

**問1** 「その違い」とは何の違いか。
1. 禁煙と罰金 2. 煙と臭い 3. 酒とタバコ 4. 酔っ払いと喫煙者

**問2** 「これ」とは何を指しているか。
1. タバコを吸うこと 2. タバコが臭く煙いこと 3. タバコの煙が周りの健康を害すること 4. タバコを吸える場所が狭いこと

## 6日目：説明文③ — 文末「ない」に注意

**ポイント：文末が「ない」で終わる表現**
- ～とは限らない、～にすぎない、～に違いない、～かねない、～わけにはいかない。

### 練習：会話文
A：この地球儀、でこぼこしているね。
B：山のところは出っ張っているんだ。でも変だよ。地球の大きさから考えると、人工衛星だって表面から1センチも離れていない。山は1センチ以上出ている。正しい地球儀はツルツルでなければ。

**問（正しいものを選ぶ）**
1. この地球儀は丸くない。
2. この地球儀の表面はツルツルしていない。
3. この地球儀は山の位置が間違っている。
4. この地球儀では山は1センチでなければいけない。
5. この地球儀は実際の山の高さを正しく表していない。

### 問題：でこぼこの地球儀
国際宇宙ステーションは地球から約400キロ上空にある。これは地球の直径から言えば1センチ程度で、表面とほとんど変わらない。海や山を表現したでこぼこの地球儀は全く正しくない。30センチの地球儀ではエベレストも0.2ミリにすぎない。正しい地球儀はツルツルであるべきだ。

**問1** 「地上と変わらないところを回っているにほかならない」とは？
1. 地表に近いところを回っているだけだ 2. 変化のないところを回っている 3. 表面を回っているわけではない 4. 地面と同じ場所を回っているわけではない

**問2** 「ツルツルでなければならない」とはどういう意味？
1. でこぼこしていてはおかしい 2. 回転するようにしなければならない 3. でこぼこが逆でないとおかしい 4. きれいに磨かなければならない

## 7日目：実戦問題

**問題1：相談者と回答者A・B**
給食費を払っているのに「いただきます」を言うのはおかしいという相談者に対し、Aは「命への感謝」と諭し、Bは「挨拶の一種」として割り切ることを勧める。

**問1** 相談者は「いただきます」をどう理解しているか。
1. 給食前の挨拶表現。 2. 人からの贈り物への感謝。 3. 給食費を払わない場合の表現。 4. 料理への感謝。

**問2** A・Bの回答について正しいのは？
1. Aは感謝を理解しないことに驚き、Bは使わないことに驚く。 2. Aは抗議を支持、Bは抗議しない方がいいとする。 3. 二人とも相談者が意味を理解すべきと言っている。 4. 二人とも家庭でも強制すべきと言っている。

**問題2：脳の構造と教育**
人間の記憶力はコンピュータに比べれば親指程度の脳の容量にすぎない。暗記ばかりを重視し、それで一生を左右するのは僭越である。

**問1** 「その答を出して」とは？
1. 成績の良し悪しを決めて 2. 記憶脳の大きさを測って 3. 仕事ができない理由を聞いて 4. 一生を左右するか考えて

**問2** 筆者が言いたいことは？
1. 脳はおぼえるだけの計算機よりすばらしい。 2. 仕事の可否は脳の部位による。 3. 脳の重要部位は小さい。 4. 暗記重視の現在の教育はおかしい。
