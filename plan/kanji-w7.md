## Day 1: 求人・募集 (Help Wanted, Recruitment)

### 1. Kanji List & Vocabulary
* **求 (7 Strokes / キュウ / もと・める):** 求人 (きゅうじん - Help wanted), 要求 (ようきゅう - Demand/request), 請求書 (せいきゅうしょ - Bill/invoice), 求める (もとめる - Demand/request).

* **単 (9 Strokes / タン):** 単語 (たんご - Word), 単位 (たんい - Unit/credit), 単に (たんに - Merely/simply), 単なる (たんなる - Mere/simple).
* **許 (11 Strokes / キョ / ゆる・す):** 許可 (きょか - Permission), 免許 (めんきょ - Permit/license), 許す (ゆるす - Permit/allow).
* **給 (12 Strokes / キュウ):** 給料 (きゅうりょう - Wage/salary), 供給 (きょうきゅう - Supply/provision), 給与 (きゅうよ - Allowance), 月給 (げっきゅう - Monthly wage), 週給 (しゅうきゅう - Weekly wage), 日給 (にっきゅう - Daily wage).
* **与 (3 Strokes / ヨ / あた・える):** 給与 (きゅうよ - Allowance), 与える (あたえる - Give/present).
* **応 (7 Strokes / オウ):** 応募 (おうぼ - Application), 応用 (おうよう - Practical use/application), 一応 (いちおう - Tentatively), 応じる (おうじる - Respond).
* **課 (15 Strokes / カ):** 課 (か - Lesson/section), 課長 (かちょう - Section chief), 日課 (にっか - Daily lesson/task).
* **程 (12 Strokes / テイ):** 課程 (かてい - Course/curriculum), 過程 (かてい - Process/course), 程度 (ていど - Degree/amount), 日程 (にってい - Schedule).
* **制 (8 Strokes / セイ):** 制度 (せいど - System), 制限 (せいげん - Restriction/limitation), 制作 (せいさく - Work/production), 体制 (たいせい - System/structure).
* **講 (17 Strokes / コウ):** 講座 (こうざ - Class/course), 講義 (こうぎ - Lecture), 講演 (こうえん - Lecture), 講師 (こうし - Lecturer).

* **導 (15 Strokes / ドウ / みちび・く):** 指導 (しどう - Leadership/guidance), 導入 (どうにゅう - Introduction), 導く (みちびく - Lead).

### 2. Guessing the Reading (読みを推測する)
* Kanjis with the same component parts often share the same On'yomi (音読み).
* **魚** (ギョ/fish) $\rightarrow$ **漁** (ギョ/fishing)
* **底** (テイ/bottom) $\rightarrow$ **低** (テイ/low)
* **告** (コク/tell) $\rightarrow$ **鉱** (コウ/mineral)
* **皮** (ヒ/skin) $\rightarrow$ **彼** (ヒ/he)
* **滴** (テキ/drop) $\rightarrow$ **適** (テキ/suitable)
* **偉** (イ/great) $\rightarrow$ **違** (イ/differ)
* **測** (ソク/measure) $\rightarrow$ **側** (ソク/side) $\rightarrow$ **則** (ソク/rule)
* **授** (ジュ/instruct) $\rightarrow$ **受** (ジュ/receive)
* **零** (レイ/zero) $\rightarrow$ **冷** (レイ/cold) $\rightarrow$ **令** (レイ/command)
