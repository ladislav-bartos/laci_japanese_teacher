## 1日目：コンサートやリサイタルのお知らせ

**ポイント：決まったパターンに慣れよう**

### Key vocabulary
- **リサイタル** — recital
- **開演／開場** — starting time / opening of the hall
- **全席自由** — free seating
- **前売(券)／当日(券)** — advance ticket / ticket sold on the day
- **入賞／デビュー／共演／公演** — prize-winning / debut / co-star / performance
- **クラシック／ジャズ／ギタリスト／ピアニスト** — classical / jazz / guitarist / pianist

### 練習：会話文
> A: 今度、大谷美和のコンサート行かない？ ギタリスト2人とピアニストの4人でやるんだって。
> B: 行く、行く。日本でのコンサートは久しぶりだよね。
> A: うん、パリに住んでいるし、海外での活動のほうがずっと多いらしいよ。
> B: ふーん。もっと日本でやってくれればいいのに。
> A: そうだよね。あ、今度はクラシックだけじゃなくて、映画音楽や日本の歌もやるんだって。

**問（正しいものを選ぶ）**
1. AさんはBさんをコンサートに誘っている。
2. 大谷美和は今度日本で初めてコンサートを開く。
3. 大谷美和は日本国内での活動が少ない。
4. 大谷美和は今度のコンサートでクラシックを演奏しない。
5. Bさんは、大谷美和のコンサートに行きたがっている。

### 問題：大谷美和 バイオリンリサイタル
- **日時：** 20XX年7月3日（土）（開場：14:30 開演15:00）
- **場所：** ヤマセホール
- **全席自由：** 前売2,500円／当日3,000円
- **曲目：** シャコンヌ、Take Five、五木の子守歌他
- **プロフィール：** 1970年東京生まれ。国際音楽コンクールで3位入賞。現在パリ在住。

**問1** 「非常に楽しみにしています」とあるが、何が楽しみなのか。
1. 大谷美和が語ること
2. 大谷美和の笑顔を見ること
3. 大谷美和が緊張すること
4. 大谷美和のコンサート

**問2** どのようなことが書かれているか。
1. コンサートの感想
2. コンサートの紹介
3. コンサートの批評
4. コンサートの批判

## 2日目：断水のお知らせ (Water Outage)

**ポイント：日時と理由をチェック！**

### Key vocabulary
- **断水** — water outage
- **日時** — date and time
- **水道工事** — water supply construction
- **くみ置き** — water stored in advance (emergency)
- **赤く濁った水** — red/cloudy water

### 練習：会話文
> 母親：今日は夜中にお風呂のお湯を捨てないで。断水だからバケツにくんで使ってね。飲み水は冷蔵庫やペットボトルにもたくさんあるから。
> 息子：もうわかったよ。今夜は早く寝るし、朝4時前になんて起きないし。
> 父親：ははは、家中水だらけだな。断水は一晩だけだっていうのに。

**問（正しいものを選ぶ）**
1. 母親は節水をしている。
2. 母親は断水をしている。
3. 母親は断水に備えて準備をしている。
4. 災害のため、水が止まってしまった。
5. 断水は今夜11時半から翌朝4時までらしい。

### 問題：断水のお知らせ（緑が丘地区）
- **作業時間：** 2月20日（木）23:30 〜 2月21日（金）4:00
- **場所：** 緑が丘1丁目大山神社付近
- **注意事項：** 工事終了後、赤く濁った水が出る場合は水道課へ連絡すること。

**問1** Aの内容と合っていないものはどれか。
1. 水道の工事があるので、必要な水は用意しておいたほうがいい。
2. 水道の工事があるのは一晩だけだ。
3. 水道の工事中は赤く濁った水が出るかもしれない。
4. 水道の工事のため4時間半ぐらい水が出ない。

**問2** AとBの組み合わせの正解はどれか。
1. ①水が一時的に出なくなる ②水を大切に使ってほしい
2. ①水が一時的に濁る ②水が出なくなるかもしれない
3. ①水を大切に使ってほしい ②水をくんでおいてほしい
4. ①水が濁るかもしれない ②水源の水位に注意してほしい

## 3日目：日帰り旅行のお知らせ

**ポイント：常識を働かせて必要な情報を読み取ろう**

### Key vocabulary
- **日帰り旅行** — day trip
- **集合時間・出発時間** — gathering time / departure time
- **研修** — training / workshop
- **参加費** — participation fee
- **定員** — capacity (maximum number of people)
- **締め切る** — to close (applications)

### 練習：会話文
> 生徒：先生、これ、海に遊びに行くっていうお知らせですよね。行きたいです！
> 教師：早く申し込みをしたほうがいいですよ。40人しか行けませんから。
> 生徒：代金は明日でもいいですか？
> 教師：申し込みだけ今日したらいいんじゃない？ 20日は海の日で休みですから。

**問（正しいものを選ぶ）**
1. 二人は旅行会社に来ている。
2. 二人は旅行のお知らせの紙を見ている。
3. 旅行に行く人は授業を休む。
4. 代金は旅行の当日に払う。
5. 参加者が40人になったら、申し込みはできなくなる。

### 問題：夏休みバス旅行のお知らせ
- **行き先：** 大島町の白浜海水浴場
- **日時：** 7月20日（月）集合 8:50（出発 9:00）
- **参加費：** 2,500円（交通費・食事代込み）
- **定員：** 40名
- **申込締切：** 7月10日（定員になり次第締め切り）

**問1** 参加者が旅行の当日、注意しなければならないことは何か。
1. お弁当など昼食の準備
2. 出発の10分前までに集まること
3. 参加費を持ってくること
4. 申し込みの定員が40名になること

**問2** このお知らせの内容と合うものはどれか。
1. 急に行けなくなったら参加費は返してもらえる。
2. 参加費のほかに交通費や食事代などがかかる。
3. 旅行先に泊まらずにその日のうちに帰ってくる。
4. 少しでも雨が降ったら旅行は中止になる。

## 4日目：納品遅延メール

**ポイント：メール形式とビジネス用語**

### Key vocabulary
- **納品／発送／配送** — delivery / dispatch / distribution
- **納期** — deadline
- **遅延** — delay
- **入荷** — arrival of goods
- **手配** — arrangement

### 練習：会話文
> A：プリンター来るの、金曜日だったよね？
> B：それが1週間遅れるんだそうです。18日の午前中…。
> A：18日って、健康診断で会社にだれもいないんじゃない？
> B：午後には戻れると思うけど、念のため19日にしてもらいましょうか。

**問（正しいものを選ぶ）**
1. プリンターは最初の予定では今日届くはずだった。
2. プリンターは19日の遅い時間に届けてもらうことにした。
3. AさんもBさんも18日の午前中は会社にいない。
4. AさんもBさんも19日は午後にしか会社にいない。
5. プリンターは最初の予定より1週間遅れて届くことになる。

### 問題：納品遅延のお詫びメール
- **内容：** メーカーの都合で入荷が遅れた。
- **変更配送日：** 5月18日（午前中）
- **対応：** もし都合が悪ければ、このメールに返信で連絡すること。

**問1** 商品の受け取りの希望について、正しくないものはどれか。
1. 5月17日以前に希望することはできない。
2. 5月17日に希望する場合、夜遅くしかできない。
3. 5月18日の午前中に希望する場合、連絡しなくてもよい。
4. 5月18日の午後に希望する場合、連絡しなければならない。

**問2** このメールの内容と合わないものはどれか。
1. 今回の商品に送料はかかっていない。
2. 商品の受け取り日時の変更はこのメールに返信する。
3. この店は水曜日以外は営業している。
4. 商品が1週間遅れるのは、このPCショップの都合である。

## 5日目：就職試験の結果通知

**ポイント：手紙の形式と選考結果**

### Key vocabulary
- **拝啓／敬具** — Dear / Sincerely (formal letter structure)
- **書類選考** — screening of application forms
- **面接** — interview
- **採用／不採用** — hire / reject
- **合格／不合格** — pass / fail

### 練習：会話文
> 田中：先輩、就活はどうですか？
> 森：一つ落ちて、一つ受かったところ。
> 田中：それ、採用通知ですか。
> 森：一次が通っただけで、これから二次の面接があるんだ。大学の補講と重なってしまった…。

**問（正しいものを選ぶ）**
1. 田中さんは森さんの後輩である。
2. 森さんは就職が決まった。
3. 森さんにとって二次試験の日は都合が悪い。
4. 森さんは補講を休んで面接を受ける。
5. 森さんは面接の日を別の日にしてもらう。

### 問題：結果通知（A：不採用、B：面接案内）
- **A（不採用通知）：** 今回は残念ながら不採用。
- **B（面接案内）：** 一次選考合格。二次面接を5月20日（木）9:30から実施。

**問1** AとBに共通して書かれていることは何か。
1. 面接試験の結果
2. 採用試験の結果
3. 面接試験の日程
4. 採用試験の日程

**問2** Bの通知の内容と合うものはどれか。
1. 二次面接に行けない人は、前日に連絡しなければならない。
2. 森さんがこの会社に就職できるかどうかはわからない。
3. 面接を受ける人は5月20日の9時までに行かなければならない。
4. この会社の面接試験は社内で行われる予定だ。

## 6日目：支払いの督促状

**ポイント：遠回しな表現に注意**

### Key vocabulary
- **入金** — payment
- **督促状** — formal demand for payment
- **法的措置** — legal action
- **行き違い** — crossed (payment sent but notice received)
- **催促** — urge / dun

### 練習：会話文
> A：これ、何？
> B：督促状じゃない！ 代金がまだだから、早く払えって。
> A：クレジットカードで払ったよ。
> B：メールを確認したほうがいいよ。僕も前にコンビニの後払いと間違えて捨ててしまったことがあるから。

**問（正しいものを選ぶ）**
1. Aさんはインターネットで買い物をした。
2. BさんはAさんに貸したお金を早く返してもらいたい。
3. Aさんはクレジットカードで代金を支払ったと思っている。
4. BさんはAさんの払い込み用紙を間違って捨ててしまった。
5. Aさんには、よく督促状が届く。

### 問題：コーヒーショップからの督促状
- **状況：** 2月7日分が3月末から6ヶ月未納。
- **対応：** 今月末までに支払いが必要。さもないと除名措置。行き違いなら無視してOK。

**問1** この文書について正しいものはどれか。
1. 会費納入のお願いという通知は初めて来た。
2. この人は3ヵ月分の会費が未納になっている。
3. このまま会費を払わなければ会員ではなくなる。
4. 会員は会費を毎月払わなければならない。

**問2** この文書を受け取った後、会費はいつまでに払わなければならないか。
1. 10月10日
2. 10月31日
3. 11月10日
4. 11月30日
