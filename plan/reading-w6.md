## 1日目：言語に関する文章

**ポイント：長い文章はスラッシュ(/)で分けて理解しよう**
- 主題(TOPIC)と文末表現に注意し、「何が・どうだ」を述べているか理解する。

### Key vocabulary
- **掲示板** (bulletin board system)
- **書き込み** (posting)
- **進化する** (to evolve)
- **猛烈なスピードで** (at a breakneck speed)
- **言語学者** (linguist)

### 練習：会話文
A先生：最近の生徒たちの会話、何を話しているのかよくわからないですよね。
B先生：本当に。インターネットの掲示板の書き込みなんか見ると、宇宙人言葉を山のように作り出していますよね。
A先生：どれだけそういう言葉が残っていくんでしょうか。たいていは消えていきますよね。
B先生：コンピューター用語なんかは、次々辞書に追加しなくてはいけなくて大変でしょうね。

**問（正しいものを選ぶ）**
1. 最近の生徒たちは難しい内容の会話をしている。
2. 若い人たちは、掲示板などで新しい言葉を使って会話している。
3. A先生は生徒たちの会話がよくわからないが、B先生はよく理解している。
4. インターネットの掲示板に宇宙人のような人が書き込みをしている。
5. B先生はコンピュータ用語は辞書に必要だと思っている。

### 問題：増え続ける医療ミス（言語の変化）
言葉は時代とともに進化する。現在は猛烈なスピードで変化している。その原因の一つはIT（PC、携帯など）。専門用語や掲示板から新しい言葉が次々と生まれ、顔文字も入り宇宙人の会話のよう。言語学者の中には、言葉は限られた数しか残らないので驚くことではないと言う人もいる。しかし、かなりの数の新しい言葉が残るのではないか。50年後の日本語は別の言語になっているかもしれない。

**問1** どうして特に驚くことではないのか？
1. 残る言葉は今までと変わらないから。 2. 生み出された言葉は思ったほど多くないから。 3. ITの影響で言葉が増えているから。 4. いつも若い人の会話は宇宙人のようだから。

**問2** 筆者が最も言いたいことは？
1. 若い人は普通の言葉で会話すべき。 2. 今後多くの新しい言葉が残るだろう。 3. 言葉は変化していない。 4. 古い日本語を大切にすべきだ。

## 2日目：化学に関する文章

**ポイント：キーワードを見つけよう**
- 何度も出てくる言葉や、カギとなる言葉を探す。

### Key vocabulary
- **シックハウス症候群** (sick-house syndrome)
- **接着剤** (adhesive)
- **ホルムアルデヒド** (formaldehyde)
- **たんぱく質** (protein)
- **アレルギー反応** (allergic reaction)

### 練習：会話文
B：この部屋に引っ越してから調子が悪いんだ。頭痛がするし、目がチカチカする。
A：シックハウス症候群じゃない？壁紙の接着剤のアレルギー症状じゃないかな。
B：そうかな。病院へ行って調べてもらおうかな。

**問（正しいものを選ぶ）**
1. Bさんは最近、新築のマンションに引っ越した。
2. Bさんの体調が悪い原因は、はっきりわかっている。
3. Bさんは引っ越しする前から接着剤に対してアレルギー症状が出ていた。
4. Aさんもシックハウス症候群である。
5. Bさんは頭痛や目がチカチカする原因を病院で調べてもらいたい。

### 問題：シックハウス症候群
新築の家に引っ越して体調が悪くなるのは、材料に対するアレルギー反応が原因。木材や壁紙の接着剤に含まれるホルムアルデヒドが主な原因。これはたんぱく質を変化させる特性がある。人体への害は当然だが、部屋の空気中のホルムアルデヒドの量はたんぱく質を変質させる量に満たないため、一概にそうとも言えない。

**問1** この文章は何について書かれたものか？
1. 症状について 2. 原因物質について 3. 治療方法について 4. 体質について

**問2** なぜ一概にそうとも言えないのか？
1. たんぱく質を変質させるとは限らないから。 2. 空気中の量が変質させるほどではないから。 3. 接着剤に含まれていない場合があるから。 4. 人によってアレルギーが違うから。

## 3日目：生物に関する文章

**ポイント：段落ごとに意味を理解しよう**
- 段落構成（事実・説明 → 筆者の意見）を意識する。

### Key vocabulary
- **スズメバチ** (hornet)
- **毒性が強い** (very poisonous)
- **アレルギー体質** (allergic constitution)
- **習性** (behavior patterns)
- **抗体** (antibody)

### 練習：会話文
男：スズメバチに刺されたことがあるけど、すごく痛かった。
女：私はアレルギー体質だから、次に刺されたらアレルギー反応を起こすかもしれない。
男：刺されないように気をつけなきゃね。

**問（正しいものを選ぶ）**
1. 2人はハチの種類について話している。
2. 2人ともハチに刺されたことがある。
3. 男の人はアレルギー体質である。
4. 女の人はハチに刺されたときにアレルギー反応を起こした。
5. 女の人はハチに刺されないように特に注意が必要だ。

### 問題：ハチの怖さ
ハチは痛い。特にスズメバチは毒性が強く、針も何度でも刺せる。アレルギーによるショック症状も怖い。2回目に刺されると抗体によりアレルギー反応を起こし、呼吸困難になることもある。予防には白っぽい服を。またカチカチと音がしたらすぐに逃げること。

**問1** 「怖いのはそれだけではない」が、何が怖いのか？
1. 刺されること 2. 痛いこと 3. アレルギー体質になること 4. アレルギー反応を起こすこと

**問2** ハチに攻撃されないためには？
1. 帽子をかぶらない。 2. 黒っぽい服を着ない。 3. 音を立てて逃げる。 4. 集団行動を避ける。

## 4日目：物理に関する文章

**ポイント：複雑な文章を整理して理解しよう**
- 「問題提起 → 結論① → 説明① → 結論② → 説明②」といった構成をつかむ。

### 練習：会話文
男：いちばん前がいちばん怖いとは限らない。頂上を通過するときは後ろの席のほうが加速して怖いんだ。
女：いちばん下を通るときは逆なの？
男：そう。後ろの車両に押されて加速が増すからね。
女：理屈ではそうかもしれないけど、前には何にもないからやっぱり一番前は怖いな。

**問（正しいものを選ぶ）**
1. 二人はジェットコースターに乗っている。
2. 女の人はジェットコースターに乗りたがっている。
3. 男の人は冷静にジェットコースターを見ている。
4. ジェットコースターのスピードはつねに変わらない。
5. ジェットコースターは座席によって怖さの感じ方が違う。

### 問題：ジェットコースターの怖さ
頂上通過時は後部車両が坂を上っているため、先頭は加速がゆるやか。逆に後部は加速している先頭に引っ張られ、速いスピードで抜ける。谷の通過時は逆に、前は加速してしまう後部に押されるため怖い。後部は上り坂で先頭が落ちているため遅い。

**問1** 「怖さが違うのでしょうか」に筆者はどう言っているか。
1. 同じ 2. 違う 3. どちらとも言えない 4. 人によって違う

**問2** 文章の内容と合うものは？
1. 怖がりは先頭車両に乗るべきでない。 2. スリルを楽しみたい人は先頭車両だ。 3. 下の部分では先頭車両の速度が増す。 4. 頂上通過時は先頭車両の速度が増す。

## 5日目：医学に関する文章

**ポイント：論理の構成（例・説明 → 結論）をつかむ**
- 文章の構成：何の話か → それは何か → 詳細説明 → 筆者の意見。

### 練習：会話文
子：お母さん、あれ何？
母：AEDっていうのよ。急病の人を助ける機械。
子：駅の人が使うの？
母：だれでも使っていいのよ。心臓が止まった人に使うみたい。練習させてほしいよね。

**問（正しいものを選ぶ）**
1. AEDは緊急時に助けを呼ぶ機械である。
2. AEDは急病人に使われる医療機器である。
3. AEDを使うには資格が必要だ。
4. この母親はAEDの使い方を知っている。
5. この母親はAEDに関心を持った。

### 問題：AEDとは
AEDは心室細動になった心臓に電気ショックを与え、正常リズムに戻す医療機器。音声ガイドがあり専門知識がなくても使える。処置は時間との勝負で、1分ごとに成功率は10%低下する。救急車到着までに平均7分かかるため、一般市民がAEDを使うことが重要。

**問1** 空欄（ ）に入るのは？
1. 10% 2. 30% 3. 50% 4. 70%

**問2** 内容と合っているものは？
1. AEDは設置場所を増やす必要がある。 2. 一般市民には難しい。 3. 使えないのが現状だ。 4. 学んでおけば一般市民でも人の命が救える。

## 6日目：数学に関する文章

**ポイント：構成（第1〜5段落）を把握する**
- ラクダの話の構造理解。第1段落：筆者の呼びかけ。第2段落：昔の話。第3段落：後日談（失敗例）。

### 練習：会話文
子：ラクダの計算の話、知ってる？借りた1頭も返せたって話。
父：マネした人が借りた1頭を返してもらえなかったっていう話ね。分数にして足し算したら答えが違うことに気づくよ。

**問（正しいものを選ぶ）**
1. 子どもはラクダの話を聞いたことがない。
2. ラクダの計算は必ずうまくいく。
3. 話には返せた話と返せない話がある。
4. 父親は子どもから教えてもらった。
5. 子どもは父親のヒントで理解した。

### 問題：ラクダの計算
昔、商人が3人の息子に「長男1/2、次男1/3、三男1/9」と遺言。計17頭だったので割り切れない。旅の老人が1頭貸して18頭にし、配分して1頭余ったので返してもらった。後日、同様に11頭で試したが、今度は合計が12頭になり損をした。これは「1/2+1/3+1/6=1」になることに気づかなかったから。

**問1** 「この話」とは？
1. 分数が苦手な人の話 2. ラクダの分配の話 3. 興味を持った人の話 4. アラビアの老人の話

**問2** 空欄（②）に入るのは？
1. 数が少なくなる 2. 数が多くなる 3. 割り切れて余りがなくなる 4. また割り切れなくなる

## 7日目：実戦問題

**問題1：震災10年記念写真展**
古川市美術館で、地震直後の町の様子を撮った写真展が開催されている。市民の安川さんは、壊れた家を少しでも残したいという思いで夢中で撮った。復興が進んだ今、当時の出来事を思い出す場となっている。

**問1** 「ここで起こった出来事」とは？
1. 地震 2. 安川さんの家が壊れたこと 3. 夫が亡くなったこと 4. 写真展

**問2** この文章の内容と合うものはどれか？
1. 安川さんは地震発生時に撮った。 2. 安川さんは10年後の町を撮った。 3. 安川さんは地震直後の町を撮った。 4. 安川さんは地震後にプロになった。

**問題2：踏切での人身事故**
横浜市港北区で、女性が列車にはねられ死亡した。女性は遮断棒を持ち上げて線路内に入ったとみられ、引きずられた跡や血痕があることから、ひき逃げ事件として捜査中。

**問1** 警察がひき逃げと考えているのはなぜか？
1. 跡があったから。 2. 飲酒運転が多発しているから。 3. うつぶせで倒れていたから。 4. 道路をわたっていたから。

**問2** この記事の内容と合うものはどれか？
1. 飲酒運転をした人が引いた。 2. 女性は身元不明。 3. 踏切事故の女性は助からなかった。 4. 運転していた男性が通報した。

**問題3：「おさかなポスト」**
外来魚が川に捨てられるのを防ぐため、川崎市で「おさかなポスト」が設置された。不況などで飼えなくなった魚が捨てられるが、管理者は命の尊さを伝えている。また、外来種が増え生態系が乱れることを防ぐ目的もある。

**問1** 両方の記事に共通する内容はどれか？
1. ポスト設置のきっかけ 2. 設置目的 3. 捨てられる理由 4. 捨てられる数

**問2** 管理者はどのように思っていると考えられるか？
1. 数が増えて困っている。 2. 捨てないでほしい。 3. もっと集めたい。 4. 生き物の命を大切にしてほしい。
