# Kanji Learning Summary - Week 1

This document extracts the essential Kanji information, vocabulary, and practice questions for Week 1, Days 1 through 7.

## Day 1: Notices and Warnings (1日目)

### 1. Kanji List & Vocabulary

* **禁 (13 Strokes / キン):** 禁止 (きんし - Prohibition), 禁煙 (きんえん - No smoking).
* **煙 (13 Strokes / エン / けむり):** 禁煙 (きんえん - No smoking), 煙 (けむり - Smoke).
* **静 (14 Strokes / セイ / しず・か):** 安静 (あんせい - Quietly), 静か (しずか - Quiet).
* **危 (6 Strokes / キ / あぶ・ない):** 危機 (きき - Crisis), 危ない (あぶない - Dangerous).
* **険 (11 Strokes / ケン / けわ・しい):** 危険 (きけん - Danger), 険しい (けわしい - Steep).
* **関 (14 Strokes / カン / かか・わる):** 関心 (かんしん - Interest), 関わる (かかわる - Have to do with).
* **係 (9 Strokes / ケイ / かかり):** 関係 (かんけい - Relation), 係員 (かかりいん - Person in charge).
* **落 (12 Strokes / ラク / お・ちる):** 転落 (てんらく - Fall), 落石 (らくせき - Falling rocks).
* **石 (5 Strokes / セキ / いし):** 磁石 (じしゃく - Magnet), 石 (いし - Stone).
* **飛 (9 Strokes / ヒ / と・ぶ):** 飛行場 (ひこうじょう - Airport), 飛び出す (とびだす - Run out suddenly).
* **駐 (15 Strokes / チュウ):** 駐車場 (ちゅうしゃじょう - Parking lot), 駐車 (ちゅうしゃ - Parking).
* **捨 (11 Strokes / シャ / す・てる):** 四捨五入 (ししゃごにゅう - Rounding), 捨てる (すてる - Throw away).
* **遊 (12 Strokes / ユウ / あそ・ぶ):** 遊泳 (ゆうえい - Swimming), 遊ぶ (あそぶ - Play).
* **泳 (8 Strokes / エイ / およ・ぐ):** 水泳 (すいえい - Swimming), 泳ぐ (およぐ - Swim).

### 2. Practice Questions

* 「関係者以外立ち入り禁止」= 関係者でない人は (**a.** 入らなければならない / **b.** 入ってはいけない)
* 「お静かに」 = (**a.** 静かにしてください / **b.** 静かにしてはいけません)
* 「(**a.** 危うい / **b.** 危ない)ですから、下がってください。」
* 「スピード落とせ」 = スピードを (**a.** 上げろ / **b.** おそくしろ)
* 「遊泳禁止」 = (**a.** ここで遊ぶべきだ / **b.** ここで泳いではいけない)
* 「落石注意」 = (**a.** 石が落ちてくるかもしれない / **b.** 石を落としてはいけない)
* 「駐車禁止」 = (**a.** 駐車場がない / **b.** 車を止めてはいけない)

## Day 2: Signs Often Seen Inside Buildings (2日目)

### 1. Kanji List & Vocabulary

* **喫 (12 Strokes / キツ):** 喫茶店 (きっさてん - Coffee shop), 喫煙所 (きつえんじょ - Smoking area).
* **非 (8 Strokes / ヒ):** 非〜 (ひ - Non-/Anti-), 非常に (ひじょうに - Extremely), 非常口 (ひじょうぐち - Emergency exit).
* **御 (12 Strokes / ゴ, おん / お):** 御手洗い (おてあらい - Toilet), 御中 (おんちゅう - Written on letters addressed to an organization).
* **常 (11 Strokes / ジョウ / つね):** 日常 (にちじょう - Everyday), 常識 (じょうしき - Common sense).
* **受 (8 Strokes / ジュ / う・ける):** 受験 (じゅけん - Taking an exam), 受ける (うける - Receive).
* **付 (5 Strokes / フ / つ・ける):** 付近 (ふきん - Vicinity), 受付 (うけつけ - Reception), 日付 (ひづけ - Date).
* **案 (10 Strokes / アン):** 案内 (あんない - Information).
* **内 (4 Strokes / ナイ / うち):** 以内 (いない - Within), 社内 (しゃない - Within the company).
* **議 (20 Strokes / ギ):** 会議室 (かいぎしつ - Meeting room), 不思議 (ふしぎ - Wonderful/strange).
* **化 (4 Strokes / カ, ケ):** 文化 (ぶんか - Culture), 化粧室 (けしょうしつ - Powder room/toilet).
* **階 (12 Strokes / カイ):** 階段 (かいだん - Stairs), 〜階 (〜かい - ...floor).
* **段 (9 Strokes / ダン):** 段階 (だんかい - Level), 手段 (しゅだん - Means/way), 石段 (いしだん - Stone step).

* 「日本語は(**a.** 日常 / **b.** 非常)会話ならできます。」
* あの人は(**a.** 常識 / **b.** 御中)がない。
* A「すみません、化粧室はどこですか。」 B「はい、(**a.** お手洗い / **b.** 案内所)は、あちらでございます。」
* A「会議室はどこですか。」 B「(**a.** 段階 / **b.** 階段)を上って、すぐ右です。」
* 「このごろ(**a.** 一段と / **b.** 石段と)寒くなってきましたね。」
* ベストセラーになった小説が映画(**a.** 化 / **b.** 案)された。

## Day 5: Signs Often Seen on Buses, Trains, etc. (5日目)
