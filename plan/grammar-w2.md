# Grammar Learning Summary - Week 2 (第2週)

**Theme: やればやるほどおぼえられる (The more you do it, the better you will get at it)**

This document extracts the grammar patterns, meanings, connection rules, example sentences, and practice questions from Week 2, organized day by day for self-testing.

# Day 1 (1日目): がんばったかいがあって (It was worth the effort)

*Opening: 努力のかいあって、希望の大学に合格した。*
*("My efforts were rewarded because I got into the university I wanted to.")*

## Pattern 1: 〜かいがある / 〜かいがあって (worth the effort)
* **Meaning:** The effort put in was rewarded; it was worth doing.
* **Connection:** Vる / Vた + かいがある / かいがあって
* **Examples:**
  * 努力のかいあって、希望の大学に合格した。 — My efforts were rewarded because I got into the university I wanted to.
  * この年まで待ったかいがあった。 — It was worth waiting until this age (to marry).

## Pattern 2: 〜かいもなく (despite the effort, in vain)
* **Meaning:** Despite putting in effort, the result was not achieved; in vain.
* **Connection:** Vる / Vた + かいもなく
* **Examples:**
  * 手術のかいもなく、愛犬が死んでしまった。 — Even though my pet dog had an operation, he died.
  * 一生懸命練習したかいがなかった。(予選で落ちた) — My hard practice was in vain (I failed the prelims).

## Pattern 3: 〜がい (worth doing)
* **Meaning:** Used to express that something is worth the effort, value, or effect of doing it.
* **Examples:**
  * やりがいのある仕事 — Work worth doing.
  * 教えがいのある生徒 — Students worth teaching.

# Day 2 (2日目): 借金してまで (Even to the point of borrowing money)

*Opening: 借金してまで海外旅行に行くなんて、私には信じられない。*
*("I can't believe he went on a trip abroad on borrowed money.")*

## Pattern 1: 〜てまで / 〜までして (even if it means...)
* **Meaning:** To do something extreme or surprising for a purpose.
* **Connection:** Vて + まで(も) / N + まで + Vて
* **Examples:**
  * 親にうそをついてまで遊びに行きたくない。 — I don't want to go if I have to lie to my parents.
  * いやな思いをしてまで、彼女と付き合うことないよ。 — There is no reason to keep going out with her if you don't enjoy it.

# Day 3 (3日目): 本日限り (Only today)

*Opening: 本日限り、安いうちに、今のうちに！*
*("Only today! Cheap now! While you still can!")*

## Pattern 1: 〜うちに (while...)
* **Meaning:** Do something while a state or condition still exists; before it changes.
* **Connection:** Vる / Vない / Vている / Aい / naな / Nの + うちに
* **Examples:**
  * 忘れないうちに、メモしておこう。 — I will make a note before I forget.
  * 桜がきれいなうちに、お花見に行きましょう。 — Let's go to see the cherry blossoms while they are still beautiful.

## Pattern 2: 〜か / 〜ないかのうちに (as soon as / just as)
* **Meaning:** Something happens almost simultaneously with another action.
* **Connection:** Vる + か + Vない + かの + うちに
* **Examples:**
  * 授業が終わるか終わらないかのうちに、彼は教室を出た。 — As soon as class ended, he left the classroom.
  * 布団に入るか入らないかのうちに眠ってしまう。 — I fell fast asleep the moment I got under the covers.

## Pattern 3: 〜限り (as long as / to the limit of)
* **Meaning:** As long as a certain condition continues; or within the scope of one's knowledge/abilities.
* **Connection:** Vる / Vない / Vている / Aい / Aくない / naな / naである / Nである + 限り
* **Examples:**
  * 日本にいる限り、日本語は必要だ。 — As long as you are in Japan, you need Japanese.
  * 私の調べた限りでは、この研究はだれもしていない。 — To the best of my research, no one has done this study.

# Day 4 (4日目): 君さえいれば (If only you were here)

*Opening: 君さえいれば幸せ。かわいいからこそ、しかることもあるんだよ。*
*("If only you were here, I'd be happy. I scold you sometimes because I care for you.")*

## Pattern 1: 〜さえ / 〜さえすれば (if only / just by)
* **Meaning:** If only X were the case; just doing X is enough.
* **Connection:** N + さえ / Vば / Aければ / naなら + さえすれば
* **Examples:**
  * 携帯電話さえあれば、時計もカメラもいらない。 — As long as I have a cell phone, I don't need a watch or camera.
  * この薬を飲みさえすれば、すぐに治ります。 — Just by taking this medicine, you will recover immediately.

## Pattern 2: 〜からこそ (precisely because)
* **Meaning:** Emphasizing that a reason is the *true* or *specific* reason.
* **Connection:** V / A / na / N + からこそ
* **Examples:**
  * 子どもがかわいいからこそ、しかるんです。 — I scold my child precisely because I love him/her.
  * 努力したからこそ、成功したんです。 — I succeeded precisely because I worked hard.

## Pattern 3: 〜てこそ (only after / only when)
* **Meaning:** Emphasizing that a certain state is a prerequisite for something else to happen.
* **Connection:** Vて + こそ
* **Examples:**
  * お互いに信頼してこそ、悩みも言える。 — Only when we trust each other can we talk about our problems.
  * 親になってこそ、親の苦労がわかる。 — Only after becoming a parent do you understand the hardships of parents.

# Day 5 (5日目): 外国へ行くとしたら (If I were to go abroad)

*Opening: 外国へ行くとしたら、勉強を目的として行きたい。*
*("If I were to go abroad, I would want to go for the purpose of studying.")*

## Pattern 1: 〜にしたら / 〜にすれば (from the perspective of)
* **Meaning:** Taking the perspective of a specific person or group.
* **Connection:** N + にしたら / にすれば
* **Examples:**
  * 子どもにしたら、親の期待は苦痛だ。 — From the child's perspective, parental expectations are painful.

## Pattern 2: 〜としたら / 〜とすれば (if we assume)
* **Meaning:** Hypothesizing a condition; "if".
* **Connection:** V / A / na / N + としたら / とすれば
* **Examples:**
  * それが本当だとしたら、夢のようです。 — If that is true, it is like a dream.

## Pattern 3: 〜としても (even if / even assuming)
* **Meaning:** Even if a certain assumption is true.
* **Connection:** V / A / na / N + としても
* **Examples:**
  * 行くとしても、旅行者としてしか行けない。 — Even if I go, I can only go as a tourist.

# Day 6 (6日目): 高くなるにしたがって (As it gets higher)

*Opening: 高くなるにしたがって空気が薄くなる。*
*("As it gets higher, the air becomes thinner.")*

## Pattern 1: 〜とともに (together with / at the same time as)
* **Meaning:** Two things happening simultaneously; or one change causing another.
* **Connection:** N + とともに / Vる + とともに
* **Examples:**
  * 家族とともに来日した。 — I came to Japan together with my family.
  * 自動車が普及するとともに、事故も増加した。 — As cars became popular, accidents increased as well.

## Pattern 2: 〜にともなって (along with / as)
* **Meaning:** As X happens, Y also happens (often causal).
* **Connection:** N + にともない / Vる + にともなって
* **Examples:**
  * 人口の増加にともない、資源の消費量も増える。 — As population increases, resource consumption increases.

## Pattern 3: 〜につれて (as / in proportion to)
* **Meaning:** A gradual, progressive change occurring in parallel with another.
* **Connection:** N + につれて / Vる + につれて
* **Examples:**
  * 年を取るにつれて、体の機能が低下する。 — As we get older, body functions deteriorate.

## Pattern 4: 〜にしたがって (in accordance with / as)
* **Meaning:** As X changes, Y changes; or following a rule/instruction.
* **Connection:** N + にしたがって / Vる + にしたがって
* **Examples:**
  * 北へ行くにしたがって、紅叶は早くなる。 — As you go north, the autumn colors come earlier.
  * 予定表にしたがい、学習を進める。 — I study according to the schedule.
