## Day 1: 速報 (News Flash / Headlines)

* **盗 (11 Strokes / トウ / ぬす・む):** 強盗 (ごうとう - Robbery), 盗難 (とうなん - Theft), 盗む (ぬすむ - Steal).

* **疑 (14 Strokes / ギ / うたが・う):** 疑問 (ぎもん - Question/doubt), 容疑 (ようぎ - Suspicion), 疑う (うたがう - Doubt/suspect).
* **捕 (10 Strokes / ホ / つか・まる, つか・まえる, と・る, とら・える):** 逮捕 (たいほ - Arrest), 捕まる (つかまる - Be caught), 捕まえる (つかまえる - Catch), 捕る (とる - Take/catch fish), 捕らえる (とらえる - Seize/grasp).
* **絶 (12 Strokes / ゼツ / た・える, た・つ):** 絶対(に) (ぜったいに - Absolutely), 絶つ (たつ - Sever/cut off), 絶えず (たえず - Constantly).
* **党 (10 Strokes / トウ):** 政党 (せいとう - Political party), 与党 (よとう - Ruling party), 野党 (やとう - Opposition party), 〜党 (〜とう - Party suffix).
* **補 (12 Strokes / ホ / おぎな・う):** 候補(者) (こうほしゃ - Candidate), 補助 (ほじょ - Assistance), 補足 (ほそく - Supplement), 補う (おぎなう - Supplement/compensate).
* **童 (12 Strokes / ドウ):** 児童 (じどう - Children/juvenile), 童話 (どうわ - Fairy tale).
* **爆 (19 Strokes / バク):** 爆発 (ばくはつ - Explosion).
* **暴 (15 Strokes / ボウ / あば・れる):** 暴走 (ぼうそう - Running wildly), 暴れる (あばれる - Behave violently), 乱暴(な) (らんぼうな - Rude/violent).
* **亡 (3 Strokes / ボウ / な・くなる, な・くす):** 死亡 (しぼう - Death), 亡くなる (なくな・る - Die), 亡くす (なく・す - Lose someone).
* **罪 (13 Strokes / ザイ / つみ):** 〜罪 (〜ざい - Crime of), 有罪 (ゆうざい - Guilty), 無罪 (むざい - Innocent), 罪 (つみ - Crime).

## Day 7: 実戦問題 (Practice Exercise)

### 1. Kanji Reading (読み方)
* **経験の有無** (**1.** あむ / **2.** ゆうむ / **3.** うむ / **4.** ゆうぶ)
* **手間を省く** (**1.** のぞく / **2.** はぶく / **3.** いだく / **4.** ほどく)
* **入居者を募集** (**1.** いりおしゃ / **2.** にゅうこしゃ / **3.** いりいしゃ / **4.** にゅうきょしゃ)
* **徒歩3分** (**1.** とぼ / **2.** そぼ / **3.** とほ / **4.** そほ)
* **富を得る** (**1.** ある / **2.** いる / **3.** える / **4.** おる)
* **財布** (**1.** とおり / **2.** へんり / **3.** まわり / **4.** あたり)
* **木綿** (**1.** こめん / **2.** きわた / **3.** もめん / **4.** こわた)
* **束ねる** (**1.** たばねて / **2.** たまねて / **3.** そこねて / **4.** そくねて)
* **処分** (**1.** こぶん / **2.** しょぶん / **3.** きょぶん / **4.** ちょぶん)
* **無事** (**1.** むごと / **2.** むじ / **3.** みごと / **4.** ぶじ)

### 2. Kanji Writing (漢字で書く)
* ふるさとは自然が **ゆたか** です。(1. 富か / 2. 豊か / 3. 財か / 4. 泉か)
* 手がすべって、**おさら** を割ってしまった。(1. 器 / 2. 皿 / 3. 甘 / 4. 宝)
* 今日は牛肉が **おかいどく** です。(1. 説 / 2. 特 / 3. 得 / 4. 読)
* 日本でいちばん大きい **みずうみ** はどこにありますか。(1. 湖 / 2. 谷 / 3. 湾 / 4. 岸)
* 1位の選手には **しょうきん** がおくられます。(1. 募られ / 2. 遅られ / 3. 層られ / 4. 贈られ)

### 3. Fill in the Blank (適当な言葉)
* この本は上中下の3 (** ) から成っている。 (1. 刊 / 2. 巻 / 3. 件 / 4. 枚)
* 今から、口 (** ) 試験があります。 (1. 面 / 2. 筆 / 3. 記 / 4. 述)
* ここにずっと住むわけではなく、(** ) の住まいです。 (1. 仮 / 2. 公 / 3. 単 / 4. 応)
* 仕事で業 (** ) を上げれば給料も上がるだろう。 (1. 責 / 2. 積 / 3. 績 / 4. 漬)
* 水 (** ) にってお茶を入れて持参する。 (1. 竹 / 2. 筒 / 3. 封 / 4. 包)
* 野 (** ) から首相の発言に対して批判の声が上がった。 (1. 党 / 2. 軍 / 3. 童 / 4. 覚)
* 国は (** ) 額の借金があります。 (1. 亡 / 2. 総 / 3. 巨 / 4. 満)

### 4. Kanji Writing Application
* 会社の長は社長、学校の長は校長、大学の長は学長で、(**1. 政府**) の長は総理 (**2. 大臣**) です。厚生労働省、文部科学省、環境省など各省の長も (**3. 大臣**) ですが、気象庁、警察庁など庁の場合は (**4. 長官**) と呼ばれます。
