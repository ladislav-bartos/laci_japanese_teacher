# Kanji Learning Summary - Week 5

This document extracts the essential Kanji information, vocabulary, and practice questions for Week 5, Days 1 through 7.

## Day 1: 家庭用品(ポット・ヒーター) (Household Goods - thermos flask, heater)

### 1. Kanji List & Vocabulary
* **傾 (13 Strokes / ケイ / かたむ・く, かたむ・ける):** 傾向 (けいこう - Trend/tendency), 傾く (かたむく - Tilt/incline), 傾ける (かたむける - Make something tilt/incline).
* **横 (15 Strokes / オウ / よこ):** 横転 (おうてん - Overturn), 横断歩道 (おうだんほどう - Pedestrian crossing), 横 (よこ - Side).
* **湯 (12 Strokes / トウ / ゆ):** 熱湯 (ねっとう - Boiling water), 湯飲み (ゆのみ - Teacup), 湯 (ゆ - Hot water), 湯気 (ゆげ - Steam).
* **恐 (10 Strokes / キョウ / おそ・れる, おそろ・しい):** 恐怖 (きょうふ - Fear), 恐ろしい (おそろしい - Fearful/terrible), 恐れ (おそれ - Fear/risk), 恐れ入ります (おそれいります - I beg your pardon/Excuse me).
* **原 (10 Strokes / ゲン / はら):** 原料 (げんりょう - Raw material), 原産 (げんさん - Origin/native), 野原 (のはら - Field).
* **因 (6 Strokes / イン):** 原因 (げんいん - Cause), 要因 (よういん - Factor).
* **置 (13 Strokes / チ / お・く):** 位置 (いち - Location), 装置 (そうち - Device), 置く (おく - Put), 物置 (ものおき - Closet/storeroom).
* **寝 (13 Strokes / シン / ね・る):** 寝室 (しんしつ - Bedroom), 寝坊 (ねぼう - Late riser/getting up late), 昼寝 (ひるね - Nap), 寝る (ねる - Sleep).
* **熱 (15 Strokes / ネツ / あつ・い):** 熱 (ねつ - Heat/fever), 熱中 (ねっちゅう - Mania/passion/enthusiasm), 熱心(な) (ねっしんな - Enthusiastic/zealous), 熱い (あつい - Hot).
* **焼 (12 Strokes / ショウ / や・く, や・ける):** 燃焼 (ねんしょう - Combustion), 焼く (やく - Roast/grill), 焼ける (やける - Be burnt/baked).
* **触 (13 Strokes / ショク / ふ・れる, さわ・る):** 接触 (せっしょく - Contact), 感触 (かんしょく - Sense of touch), 触れる (ふれる - Touch/mention), 触る (さわる - Touch).
* **灯 (6 Strokes / トウ):** 灯油 (とうゆ - Kerosene), 蛍光灯 (けいこうとう - Fluorescent light), 電灯 (でんとう - Electric light), 灯台 (とうだい - Lighthouse).
* **油 (8 Strokes / ユ / あぶら):** 石油 (せきゆ - Oil/petroleum), 原油 (げんゆ - Crude oil), 油断 (ゆだん - Negligence/inattention), 油 (あぶら - Oil).
* **余 (7 Strokes / ヨ / あま・る):** 余分(な) (よぶんな - Excessive), 余計(な) (よけいな - Excessive/needless), 余裕 (よゆう - Leeway/time to spare), 余る (あまる - Be left over).

### 2. Practice Questions
* 試験に出される問題の (**a.** 傾向 / **b.** 原因) を調べて受験勉強をする。
* 「(**a.** 恐れ入ります / **b.** 恐ろしいです) が、荷物はそちらに置いてください。」
* アフリカ (**a.** 原料 / **b.** 原産) のお茶を飲む。
* この車にはABS という安全 (**a.** 装置 / **b.** 位置) がついています。
* 「作品には手を (**a.** 触れないで / **b.** 届けないで) ください。」
* 今朝は寝坊をして朝食をとる (**a.** 余分 / **b.** 余裕) がなかった。
* 電灯をつけるのを忘れるぐらいゲームに (**a.** 熱心 / **b.** 熱中) した。
* (**a.** 灯台 / **b.** 灯油) は船に位置を教えます。
