## 1日目：エッセイ① — 答えの場所を探す

**ポイント：問いを先に読むと、早く答えが見つけられる！**
- 問いの中のキーワード（OO、ΔΔ、XX）を探すと、その近くに答えがある。

### 練習：会話文
娘：最近、本を読まなくなったなあ、私。
母：え？ いつも読んでいるじゃないの。
娘：仕事関係の本ばかりなの。必要にせまられてっていうのは違うでしょ。中学生のころからかな、こんなふうになったのは。子どものころは本当にただ本を読んでいたのに。楽しかったなあ…。
母：そうね、うれしそうに、よく図書館に通っていたわね。

**問（正しいものを選ぶ）**
1. 娘は大人になってから本を読んでいない。
2. 母は娘が子どものころ、よく本を買ってやった。
3. 娘は中学のころ、よく好きな本を読んで楽しんだ。
4. 母は、娘が子どものころ、図書館で働いていた。
5. 娘は子どものころの読書が楽しかったと思っている。

### 問題：読書
子どものころ、よく図書館へ行って本を借りたものだ。自分では楽しんで本を読んでいるという意識はなかった。読み終えた本を返しに行くと、そこに当然だが未知の本があり、それをまた借りるということを繰り返していたのだ。中学に入学したころからは、学校のレポート提出などのためにしか本を読まなくなった。社会人になってからは、仕事がらみで必要にせまられてのことが多い。今思うと、図書館通いをしていたあのころが、ほんとうに読書を楽しんでいたと言えるのかもしれない。

**問1** 「一種の作業のようなもの」とは何のことか。
1. 社会人になって本を買うこと 2. 学校でレポートを提出すること 3. レポート提出のために本を読むこと 4. 図書館で本を借りたり返したりすること

**問2** 筆者がほんとうに読書を楽しんでいたと思われるのはいつか。
1. 子どものころ 2. 子どものときからずっと今も 3. 中学生のころ 4. 仕事をするようになってから

## 2日目：エッセイ② — 実際には起きていないこと

**ポイント：過去の仮定（〜したら、〜ただろう）に注意！**
- 「もし～していたら、～ただろう」は、**実際にはしなかった／なっていない**ことを表す。

### 練習：会話文
学生：日本人はくしゃみしても「Bless you」と言わないんですね。
教師：沖縄では子どもがくしゃみをすると、そばの人が何か言うらしいですよ。幽霊に連れて行かれるとか。
学生：日本語にもそんな表現があったんですね。
教師：沖縄だけでなく、『徒然草』という古い本にも書いてあるそうです。「くさめ」と言わないと死んでしまう、と。

**問（正しいものを選ぶ）**
1. 日本では、くしゃみをすると「だれかがうわさをしている」と言わなければならない。
2. 日本には「Bless you」に当たる表現はない。
3. 昔、日本ではくしゃみをすると悪いことが起こると信じられていた。
4. 昔だけでなく今でも沖縄では、くしゃみをすると悪いことが起こる。
5. この先生も最近まで、沖縄のくしゃみについての習慣を知らなかった。

### 問題：クシャミの言い伝え
もし「日本人はくしゃみをしても何も言わないんですか」と聞かれたら、昔は「そんな習慣はない」と答えていただろう。英語の「Bless you!」のようなものがないと思っていたからだ。しかし、沖縄ではくしゃみをすると「クスクェー」と言う習慣があり、幽霊に連れていかれるのを防ぐためだそうだ。『徒然草』にも「くさめ」と言わないと死ぬという記述がある。日本人が日本の古語を知らないのは恥ずかしい。

**問1** 下線部①の質問に関して、正しいものはどれか。
1. 外国人が日本人によくする質問。 2. 筆者が日本語学校でされた質問。 3. 筆者は「習慣はない」と答えた。 4. 筆者はまだされたことがない。

**問2** この文章の内容と合うものはどれか。
1. 沖縄では英語と同じ表現がある。 2. 沖縄の言葉で14世紀の古い日本語がわかる。 3. 「くさめくさめ」は「Bless you!」にあたる。 4. 昔の日本人は病気や災害のときに「くさめくさめ」と祈った。

**問3** 筆者はどんな気持ちで下線部②のように言ったのか？
1. 日本人だからといって日本語をよく知っているとは限らない。 2. 日本人でなければ日本語のニュアンスはわからない。 3. 古語や方言を知らずに日本語を教えるのは不可能。 4. 日本人は実は日本語を知らないので注意が必要。

## 3日目：エッセイ③ — 二重否定

**ポイント：二重否定（～ないことはない、～ないわけにはいかない）は「肯定」になる！**

### 練習：会話文
店員：1円玉、たまってきたので銀行で両替してきましょうか。
店長：手数料がかかるからやめようよ。100枚両替するのに200円払わないといけないんだよ。
店員：手数料のほうが高いんですか。そんなばかな…。
店長：そのうち口座の管理料を取るかもしれないね。

**問（正しいものを選ぶ）**
1. 店長は両替することに反対している。
2. 1円玉を両替するのは手続きが面倒である。
3. 毎回、両替額より手数料のほうが高くつく。
4. 銀行にお金を預けておいても減っていく場合がある。
5. 店長は近くの銀行に口座管理料を支払っている。

### 問題：銀行の手数料
テレビショッピングの代金を振り込みに行ったら、手数料が840円もかかった。銀行の手数料は気がつかないうちに高くなっている。両替額より手数料のほうが高い場合すらある。払いたくないが、払わないわけにはいかない。最近は管理料を取る銀行もある。自分の残高がいつのまにかゼロになることもありえるだろう。今の通帳の残高では、そうならないとは言えない。

**問1** 「そうならないとは言えない」の最も近い意味は？
1. 必ずそうなる 2. そうなるかもしれない 3. そうなるかわからない 4. 絶対にそうならない

**問2** 空欄②に入る言葉は？
1. 口座を持ちたくない 2. 管理料まで払いたくない 3. 残高を知りたくない 4. 振り込みはしたくない

## 4日目：エッセイ④ — カタカナ語（日本語由来）

**ポイント：カタカナ＝外来語とは限らない！**
- 音や様子（ツルツル）、習慣（ゴミ、マンガ）、漢字が難しい（バラ、ハチ）などの理由でカタカナ表記される言葉がある。

### 練習：会話文
A：あのベンチがあるところ、ゴミだらけだと思わない？
B：みんな捨てていくよね。うちのクラブでゴミ箱作って置くっていうのはどう？ 美術部に絵をかいてもらって。
A：それ、いいかも。

**問（正しいものを選ぶ）**
1. AさんとBさんは同じクラブのメンバーである。
2. Bさんは美術部のメンバーである。
3. 学校のベンチの周りは、ゴミでいっぱいである。
4. Aさんたちは自分たちでゴミ箱を作るつもりである。
5. ベンチにかく絵を美術部に頼む予定である。

### 問題：手作りのゴミ箱
駅へ行く道にベンチがある。そこは以前、高校生たちが捨てたジュースの空き缶や弁当のゴミが散らかっていた。ある日、木でペンキを塗った手作りのゴミ箱が4個置かれた。そのゴミ箱が置かれてからは、前よりも人が集まっている。高校生たちが作ったと聞いて、心が温まった。

**問1** 「前よりも人が集まってきています」が、なぜか？
1. ゴミを捨てるため。 2. 清潔で気持ちのいい場所になったから。 3. 高校生たちの態度が良くなったから。 4. タバコを吸う場所ができたから。

**問2** 「そんな高校生」とはどんな高校生のこと？
1. ゴミ箱を作った生徒たち 2. ベンチでおしゃべりをする生徒たち 3. 心がやさしい生徒たち 4. 態度が悪い生徒たち

## 5日目：小説① — 指示語に注意（後方参照）

**ポイント：指示語が「後ろにある言葉」を指すことがある！**
- 「この人」「そういう者」などが、文中で後に説明されるパターンに注意。

### 練習：会話文（芥川龍之介「少年」より）
宣教師：お嬢さんはおいくつですか？
少女：来年12。
宣教師：今日が何日かご存知ですか？
少女：12月25日でしょう。
宣教師：何の日ですか？ ご存知ならば言ってごらんなさい。
少女：今日はお誕生日。

**問（正しいものを選ぶ）**
1. 宣教師と少女はとても親しい。
2. 少女はこれから教会へ行く。
3. 今日はクリスマスである。
4. 少女は今日が何日か知らなかった。
5. 宣教師は少女とクリスマスについて話したかった。

## 6日目：小説② — 情景と気持ち

**ポイント：登場人物の様子や気持ちを読み取る！**
- 「まっ赤になって（恥ずかしさ・怒り）」「もじもじ（緊張・迷い）」といった様子から心情を理解する。

### 練習：会話文
娘：「銀河鉄道の夜」読んだことある？
母：宮沢賢治の？ あるよ。
娘：今日読んだけど、カンパネルラって最後、死んじゃったの？
母：主人公の友だちでしょ？ かわいそうだよね。
娘：そうそう。ジョバンニ、貧乏だからっていじめられてるよね。…カンパネルラだけは優しいのに、ひどい！

**問（正しいものを選ぶ）**
1. 娘は「銀河鉄道の夜」という本を読んだことがない。
2. カンパネルラとジョバンニは本の中の登場人物である。
3. カンパネルラはジョバンニの友だちである。
4. ジョバンニは病気である。
5. カンパネルラはひどい人だ。

### 問題：「銀河鉄道の夜」
カムパネルラは先生に質問されても答えられなかった。ジョバンニは答えを知っていたが、恥ずかしさや気後れで答えることができなかった。あとで、ジョバンニはカムパネルラも同じ答えを知っていたのに、わざと答えなかったのだと気づく。カムパネルラはジョバンニのつらい状況を思いやって、わざと返事をしなかったのだ。そう考えるとジョバンニは、カムパネルラも自分もあわれな気がした。

**問1** 「手をあげようとして、急いでそのままやめました」のはなぜ？
1. 急に眠くなった 2. カンパネルラが手をあげた 3. 答えに自信がなくなった 4. 雑誌が自分のでない

**問2** 「涙がいっぱいになりました」のはなぜ？
1. 答えられなくて悔しかった 2. 自分も友だちもかわいそうだと思った 3. 仕事がつらくて悲しかった 4. 誰にもわかってもらえなくて寂しかった

## 7日目：実戦問題 (Practice Exercise)

**問題1：新幹線の隣人**
筆者は人並みに中年男は嫌いだが、新幹線で隣に座ったのが中年男でうれしかった。なぜなら、女なら仕事に集中できず、子どもはうるさく、体育会系大学生は騒ぎ、老人は激しく咳をして気が気でなかったからだ。中年男は歓迎すべき隣人である。

**問1** 「そういう希望」とは？
1. 中年男に座ってほしい 2. 隣に女に座ってほしい 3. 子どもに座ってほしくない 4. 隣にだれも座ってほしくない

**問2** なぜ喫煙車両に座ることにしたのか？
1. タバコを吸うため 2. 隣に女が座るかもしれないから 3. 年齢の近い人が多いから 4. 禁煙車両は子どもがうるさいから

**問3** なぜ中年男は歓迎すべき隣人か？
1. 暴力団である可能性が少ない 2. 静かにしている 3. 老人より健康 4. 仲良くできる

**問4** 筆者はぐっすり眠れたことについてどう思っている？
1. よかった 2. よかった（うるさかったけど） 3. 眠れたけど中年男はいやだ 4. もっと静かにしてほしかった

**問題2：百貨店とファストファッション**

**問1** AとBの共通点は？
1. 百貨店とファストファッション両方の良さ 2. 人気継続と百貨店消滅の関係 3. 生き残りにはファストファッションが必要 4. 人気継続への疑問

1. 両者とも不況が関係していると思う 2. 両者とも価値観は正反対だ 3. Aは客の視点、Bは店の視点 4. Aは否定的、Bは好意的
