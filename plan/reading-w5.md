## 1日目：見出し (Headlines)

**ポイント：見出しから文を作って理解しよう**

### Key vocabulary
- **延期** — postponement
- **収益悪化** — deterioration of revenue
- **開通** — opening (of a line/road)
- **裁判員裁判** — lay judge system/jury trial
- **自宅放火** — arson at one's own home
- **懲役** — imprisonment with work

### 練習：会話文
男：空港で寝袋にくるまって飛行機を待ってた人たち、やっと帰れるのね。
女：1週間よ。気の毒に。自然災害じゃしょうがないね。
男：乗客だけでなく商品も足止めされて航空業界の損失は2,000億〜3,000億円らしい。

**問（正しいものを選ぶ）**
1. 空港で催し物があったので、観光客は出発を遅らせた。
2. 自然災害の影響で予定の飛行機に乗れない人たちがいた。
3. 乱れていた空のダイヤはまだもとに戻らない。
4. この災害で観光客だけでなく輸出や輸入の商品も影響を受けた。
5. この災害の影響でホテル業界は多額の利益を得た。

### 問題：A国 火山噴火
A国火山噴火による影響で、欧州の航空輸送網が大混乱となった。22日中に正常化する見通し。10万便以上が欠航し、700万人以上が足止めされた。航空業界の損失は15億〜25億ユーロに上るとみられ、観光業を含めるとさらに拡大する。

**問1** 「空の足」とは何か。
1. 観光業 2. 空港の構内 3. 航空業界 4. 飛行機の便

**問2** この記事の内容と合っているものはどれか。
1. 航空・観光業界の損失は25億ユーロにとどまった。 2. 火山が噴火するといつも飛行機は飛べなくなる。 3. 火山噴火で観光客はアジアへ来た。 4. 混乱から正常化まで1週間かかった。

## 2日目：見出し② (Headlines 2)

**ポイント：助詞に注意して文の意味を予想しよう**
- 「〜も検討されている」「〜という方向へ動いている」「〜すべきだ」などの文末を推測する。

### 練習：会話文
客：お中元の商品が並んでるのね。
店員：今年は「エコ」をテーマに当店でしかお求めになれないギフトをご用意しております。
客：エコだなんて、安いものという感じがしない？
店員：環境に配慮し、産地や素材、製法などが特別なものという意味でございます。

**問（正しいものを選ぶ）**
1. エコというのは経済的だという意味だ。
2. エコというのは環境に配慮するという意味だ。
3. エコをテーマにした商品はどこでも買うことができる。
4. この店では今年は「エコ」をテーマにしたギフトを用意している。
5. この店では素材や製法にかかる費用をおさえて商品を買いやすい値段にしている。

### 問題：お中元もエコを主力に
百貨店の○○と△△はお中元商戦を前に「エコ」を強調した商品2500品を公開した。○○は有機農法や食材を餌にした豚肉など。△△は農薬を減らした野菜の冷製スープなどを主力としている。両百貨店とも「ほかでは買えない」ものを演出し、客に財布のひもをゆるめてもらうのがねらいだ。

**問1** 目玉商品と同じ意味の言葉は？
1. 中元用商品 2. エコ商品 3. ギフト商品 4. 主力商品

**問2** この記事の内容と合っているものはどれか。
1. どちらも買いやすい値段設定にしている。 2. どちらも環境に配慮した商品に注目している。 3. どちらも同じような商品を並べている。 4. 百貨店ごとに品数を変えて商戦に臨んでいる。

## 3日目：事件や事故の記事 (Crimes and Accidents)

**ポイント：定型表現を覚える**
- **所持**（possession）、**譲渡**（transfer）、**容疑**（suspicion）、**逮捕**（arrest）、**認めている**（admit）、**ひき逃げ**（hit-and-run）、**通報**（report/call to police）。

### 練習：会話文
A：またひき逃げだって。多いね。
B：どうして逃げるのかな。すぐに通報すれば助かったかもしれないのに。
A：怖くなって逃げるとか、飲酒運転していたかもしれないね。
B：飲酒運転の罰則が厳しいから、お酒を飲んでいたことがわからなくなるように逃げるんだよね。

**問（正しいものを選ぶ）**
1. Aさんは飲酒運転による交通事故の記事を読んでいる。
2. Aさんはひき逃げで幼児3人が死亡した記事を読んでいる。
3. Bさんはひき逃げをする人の気持ちが理解できない。
4. AさんとBさんは飲酒運転の罰則をもっと厳しくすべきだと言っている。
5. AさんとBさんはひき逃げ事件の罰則をもっと厳しくすべきだと言っている。

### 問題：ひき逃げか 女性死亡
18日午後9時ごろ、横浜市港北区の横浜線の踏切付近で、女性が血を流し倒れているのを男性が見つけ110番通報した。女性は死亡が確認された。付近に引きずられた跡や血痕が残っていたことから、ひき逃げ事件とみて捜査している。女性は60代ぐらいとみられ、身元確認を進めている。

**問1** 警察がひき逃げ事件と考えているのはなぜか。
1. 跡があったから。 2. 飲酒運転が多発しているから。 3. うつぶせで倒れていたから。 4. 道路をわたっていたから。

**問2** この記事の内容と合っているものはどれか。
1. 飲酒運転をした人が引いたとみられる。 2. 女性の身元はまだ不明である。 3. 踏切事故にあった女性は助からなかった。 4. 運転していた男性が通報した。

## 4日目：グラフ① (Graphs 1)

**ポイント：グラフの違いを見つける**
- 項目ごとの割合、高い・低い、グラフが2つ以上ある時の比較。

### 練習：会話文
母親：携帯電話で見るから、時計は要らないの。
息子：若い子みたいだね。
母親：でも、携帯は落としたり忘れたりして面倒なの。携帯が時計みたいに腕につけられるといいんだけど。
息子：通信機能がついた時計があるみたいだよ。
母親：便利ね。ついでに万歩計や健康診断機能もつけてほしいわ。

**問（正しいものを選ぶ）**
1. 息子は腕時計が好きだ。
2. 母親は腕時計が面倒だから使わない。
3. 息子の時計には通信機能がついている。
4. 母親は携帯電話を時計代わりにしている。
5. 現在、時計に通信機能がついているものはない。

### 問題：腕時計の付加機能
調査で、腕時計に付加機能はいらないと答えた人が7割いた。「操作が複雑になる」「デザイン性が損なわれる」という理由から。50代からは通信機能や健康診断機能、20代からはデジカメ機能が支持されている。

**問1** この記事の内容と合うグラフはどれか。（※グラフ参照）

## 5日目：グラフ② (Graphs 2)

**ポイント：グラフの表現（占める、達する、上回る、下回る）**

### 練習：会話文
妻：ライターの火事、子どもたちが犠牲になったって。かわいそうに。親が管理しないとね。
夫：昔はこんな火事なかった気がするな。車の中は燃えやすい素材が多いんだろうね。
妻：簡単に火がつかないライターにすべきよね。
夫：そういう製品もあったけど、コストが高く使いにくくて売れなかったんだって。

**問（正しいものを選ぶ）**
1. 子どもがライターをいじって起きる火事が続いている。
2. 車の中は物が燃えやすく、火が広がりやすい。
3. 昔に比べて今は、車の中にライターを置く人が多い。
4. 子どもが簡単に使えないようなライターが値上がりしている。
5. 子どもが簡単に操作できない安全基準が設けられた。

### 問題：ライターと火遊び火災
火遊びによる火災のうち発火源がライターであるものの占める割合は53%。死傷者の発生率は5歳未満で69.2%と非常に高い。

**問1** グラフの説明として正しいものはどれか。
1. 12歳以下による火災は約1300件発生し、その約4割が12歳以下である。（※グラフ参照）
2. 5歳未満の死傷者発生率は約半数を占めている。
3. 11年から20年にかけて全国ではライター火災が7割を占めている。
4. 11年から20年にかけて全国ではライター火災はその他を上回っている。

## 6日目：書評 (Book Review)

**ポイント：引用符「」『』に注意！**
- タイトルや、他人の言葉の引用、強調したい言葉に使われる。

### 練習：会話文
女：土屋先生の『人間の定義』って本、読んだ？
男：哲学の本じゃないんだ。
女：哲学を考えさせられる不思議なエッセイなの。文章は難しくないからすらすら読めて楽しかった。

**問（正しいものを選ぶ）**
1. 女の人は哲学の本を読んでいる。
2. 土屋先生のエッセイはおもしろい。
3. 土屋先生のエッセイは不思議でわかりにくい。
4. 女の人は土屋先生のエッセイを簡単に読めた。
5. 男の人は、哲学の本は読みたくない。

### 問題：『われ笑う、ゆえにわれあり』
著者はユーモアエッセイと称しているが、鋭い社会批評や哲学の問題が含まれる。非常に読みやすく、論理を考えさせられる。中でも「人間を定義するのは不可能である」という項目は笑いながら定義とは何かが見えてくる。

**問1** なぜ不思議なエッセイなのか？
1. エッセイなのに哲学のことばかりだから。 2. やさしい言葉や表現だから。 3. 知らないうちに哲学的なことを考えさせられるから。 4. 理解しにくい冗談が多いから。

**問2** 空欄②に入る言葉は？
1. ユーモアに興味がない若者 2. 哲学に興味がない若者 3. エッセイに興味がない若者 4. 条件に興味がない若者

## 7日目：実戦問題 (Practice Exercise)

**問題1：震災10年記念写真展**
古川市美術館で、地震直後の町の様子を撮った写真展が開催されている。市民の安川さんは、壊れた家を少しでも残したいという思いで夢中で撮った。復興が進んだ今、当時の出来事を思い出す場となっている。

**問1** 「ここで起こった出来事」とは？
1. 地震 2. 安川さんの家が壊れたこと 3. 夫が亡くなったこと 4. 写真展

**問2** この文章の内容と合うものは？
1. 安川さんは地震発生時に撮った。 2. 安川さんは10年後の町を撮った。 3. 安川さんは地震直後の町を撮った。 4. 安川さんは地震後にプロになった。

**問題2：踏切での人身事故**
横浜市港北区で、女性が列車にはねられ死亡した。女性は遮断棒を持ち上げて線路内に入ったとみられ、引きずられた跡や血痕があることから、ひき逃げ事件として捜査中。

**問1** 警察がひき逃げと考えているのはなぜ？
1. 跡があったから。 2. 飲酒運転が多発しているから。 3. うつぶせで倒れていたから。 4. 道路をわたっていたから。

**問2** この記事の内容と合うものはどれか。
1. 飲酒運転をしていた人がひいた。 2. 女性は身元不明。 3. 踏切事故の女性は助からなかった。 4. 運転していた男性が通報した。

**問題3：「おさかなポスト」**
外来魚が川に捨てられるのを防ぐため、川崎市で「おさかなポスト」が設置された。不況などで飼えなくなった魚が捨てられるが、管理者は命の尊さを伝えている。また、外来種が増え生態系が乱れることを防ぐ目的もある。

**問1** 両方の記事に共通する内容は？
1. ポスト設置のきっかけ 2. 設置目的 3. 捨てられる理由 4. 捨てられる数

**問2** 管理者はどのように思っているか？
1. 数が増えて困っている。 2. 捨てないでほしい。 3. もっと集めたい。 4. 生き物の命を大切にしてほしい。
